This is the API

deleteTable

delete a cloud table

URL

http://api.cloudboost.io/app/{app_id}/{table_name}

HTTP Method

DELETE

Response Type

cloudTable

Parameters

  • Path: table_name String

    name of cloud table

  • Path: app_id String

    cloud app id

  • Body: body CloudTableSave(CloudTableSave)

    master key and app id

get

get a table

URL

http://api.cloudboost.io/app/{app_id}/{table_name}

HTTP Method

POST

Response Type

cloudTable

Parameters

  • Path: table_name String

    name of cloud table

  • Path: app_id String

    id of app

  • Body: body KeyBody(KeyBody)

    master key and app id

Parameters
