delete

deletes a cloud object from cloudapp

URL

http://api.cloudboost.io/data/{app_id}/{table_name}

HTTP Method

DELETE

Response Type

CloudObject

Parameters

  • Path: app_id String

    CloudApp id

  • Path: table_name String

    cloud table name

  • Body: body CloudObjectSaveRequest(CloudObjectSaveRequest)

    CloudObject that needs to be added to the cloudboost
