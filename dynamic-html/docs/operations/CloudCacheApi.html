This is the API

clearCache

clear all items from a cloud cache

URL

http://api.cloudboost.io/cache/{app_id}/{cache_name}/clear

HTTP Method

DELETE

Response Type

List

Parameters

  • Path: app_id String

    Id of CloudApp

  • Path: cache_name String

    Name of cache

  • Body: body KeyBody(KeyBody)

    master key of app
