addMessage

URL

http://api.cloudboost.io/queue/{app_id}/{queue_name}/message

HTTP Method

PUT

Response Type

List

Parameters

  • Path: app_id String

    Id of CloudApp

  • Path: queue_name String

    Name of queue

  • Body: body QueueBody(QueueBody)

    client key of app
